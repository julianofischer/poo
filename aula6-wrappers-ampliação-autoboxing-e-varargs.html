Ampliação

    É o processo de colocar um tipo primitivo menor em um tipo primitivo maior (ou ampliado)

byte b = 1;
short s = b;
long l = s;
float f  = 1.0f;
double d = f;

Autoboxing

    É o processo de tornar um tipo primitivo em uma classe Wrapper e vice-versa

Não é possível a ampliação de um wrapper.


int a = 5;
Integer x = 5;
x = a;
Double d = 5.0;
Float f = 5.0f;
d = f; //erro

Varargs

    Permite fazer com que um método de uma classe recebe um número variável de parâmetros do mesmo tipo
    
  public void metodoTeste(int... a) {  
    /*
        a implementacao tratara da variavel 'a'
        como um array, tendo ou nao valores... 
     */
    for (int i = 0; i < a.length; i++) {
        int j = a[i];
        
    }
  }
						

  class AddVarargs {
      public void go(int x, int y) {
          System.out.println("int,int");
      }
      public void go(byte... x) {
          System.out.println("byte...");
      }
      public static void main(String[] args) {
          byte b = 5;
          AddVarargs eo = new AddVarargs();
          eo.go(b,b);

      }
  }

A AMPLIAÇÃO tem precedência sob o VarArgs


"int,int"

Resumo

  • Não é possível realizar a ampliação de um wrapper
  • Ampliação tem precedência sob boxing
  • Ampliação tem precedência sob varargs

Exercício 1


		class Varargs {
		    public void go(long... x) {
		        System.out.println("long...");
		    }
		    public void go(Long... x) {
		        System.out.println("Long...");
		    }
		    public static void main(String[] args) {
		        int b = 5;
		        Varargs eo = new Varargs();
		        eo.go(b,b);
			//o que será impresso?
		    }
		}

Exercício 2


		public class A {
		  public void t(){
		    System.out.println("A");
		  }
		}
		public class B extends A {
		  public void t() {
		    System.out.println("B");
		  }
		}
		public class C {
		  public void t(A c){
		    System.out.println("Classe C metodo A");
		    c.t();
		  }
		  public void t(B c){
		    System.out.println("Classe C metodo B");
		    c.t();
		  }
		}
		A c = new B();
		C c2 = new C();
		c2.t(c);
