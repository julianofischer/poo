Classe Object

É a raíz da hierarquia de classes em Java. Toda classe é uma subclasse de Object.

Mesmo se não usarmos "extends Object"

getClass()

  • Retorna a classe do objeto (em tempo de execução)

equals()

  • Indica se algum outro objeto é igual a este

clone()

  • Retorna uma cópia deste objeto

equals()

  • Indica se algum outro objeto é igual a este

toString()

  • Retorna uma String que é uma representação do objeto

hashCode()

  • Retorna um inteiro único de cada objeto. Veja HashMap.
