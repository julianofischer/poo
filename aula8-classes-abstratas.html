Classe Abstrata

É uma classe que não pode ser instanciada, mas pode ser superclasse de outras classes. Uma classe abstrata pode conter métodos abstratos.

Método Abstrato

Um método abstrato é um método que é declarado sem uma implementação, ou seja, não contém chaves e é seguido por um ;

						abstract void fala();

Classes Abstratas

  • Funcionam como um "molde".
  • Declaradas com a palavra chave abstract
  • Podem conter métodos abstratos
  • Podem conter métodos concretos
  • Seu propósito não é o de ser diretamente instanciada, mas de servir como superclasse modelo para outras classes

public abstract class FormaGeometrica{
    private String cor;
    public abstract double calcularArea();

    public String getCor(){
        return cor;
    }

    public void setCor(String c){
        cor = c;
    }
}

Classes Abstratas

  • Podemos criar subclasses da classe FormaGeometrica.
  • Nas subclasse é obrigatória a implementação dos métodos abstratos
  • Com isso, temos certeza que toda subclasse de FormaGeometrica implementa o método calcularArea
  • Uma classe que não é abstrata é chamada de concreta

public class Retangulo extends 
                       FiguraGeometrica{
    private double base;
    private double altura;

     public double calcularArea(){
     	return base * altura;
     }
}

Exercício 1

  • Implemente uma hierarquia de operações matemáticas
  • O programa deve conter as classes:
    • Operação Matemática (abstrata)
    • Soma
    • Subtração
    • Multiplicação
    • Divisão
  • Todas as classes devem conter o método calcula

Exercício 2

  • Implemente uma hierarquia de classes que represente os funcionários de um escritório
  • O programa deve conter as classes: Gerente, Assistente e Vendedor
  • A classe Funcionário deve conter um método abstrato double calcSalario
  • A classe Funcionário deve conter os atributos nome, matrícula e salárioBase
  • Os salários são calculados da seguinte maneira
    • Gerentes recebem 2 vezes o salárioBase
    • Assistentes recebem o salárioBase
    • Vendedores recebem o salárioBase mais uma comissão que deve ser passada por parâmetro ao método construtor
